# RSVP-app

RSVP app for an event.

## Run locally

At the moment, this repository only contains documentation and does not include runnable app code yet (no `package.json`, source files, or start script).

### Current setup steps

1. Clone the repository:
   ```bash
   git clone https://github.com/theroshanchaudhari/RSVP-app.git
   ```
2. Go to the project folder:
   ```bash
   cd RSVP-app
   ```
3. Confirm available files:
   ```bash
   ls -la
   ```

Once the application code is added, update this README with install and start commands for the chosen stack.
