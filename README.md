# RSVP-app

Reusable RSVP app for events with personalized invite links.

## Run locally

Open `index.html` in a browser.

## Personalized invite links

Generate links from the app, or build manually with query params:

- `guest`
- `event`
- `date` (optional)
- `host`

Example:

`index.html?guest=Roshan&event=Launch%20Party&date=2026-12-01&host=Team`

When someone opens their link, the page is personalized and they can submit an RSVP.
